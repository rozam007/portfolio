# Muhammad Rozam - Portfolio Website

A clean, classic, and aesthetically appealing React.js portfolio website showcasing professional experience, skills, projects, and achievements.

## Features

- **Multi-route Navigation**: Resume content split across multiple pages
  - Home/Profile
  - Experience
  - Education
  - Skills
  - Projects
  - Certifications

- **Classic Design**: Simple, professional, and aesthetically pleasing UI
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Subtle fade-in animations for better user experience

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
